# PORTES PROBABILITAT "Monty-Hall"
Una adaptació feta en HTML, JavaScript y CSS per dos estudiants de 2n de Batxillerat de Ciències del problema de les portes de Monty Hall.
### Què és el problema de les portes de Monty Hall?
El problema de les portes de Monty Hall és un problema clàssic de probabilitat condicionada que descriu una situació en què un concursant ha d’escollir entre tres portes, darrere d’una de les quals hi ha un premi i darrere de les altres dues no. Després de la primera elecció, el presentador (inspirat en el programa ```Let's Make a Deal``` i el seu conductor Monty Hall) obre una de les portes no escollides que conté un premi fals (per exemple, una cabra) i ofereix la possibilitat de canviar la decisió inicial.

La definició del problema posa de manifest que, tot i que aparentment queden dues opcions iguals, la probabilitat de guanyar és major si es canvia de porta ``(2/3)`` que si es manté l’elecció inicial ```(1/3)```, ja que l’acció del presentador està condicionada pel seu coneixement previ.
